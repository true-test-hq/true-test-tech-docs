# Модуль тестирования участника

Этот модуль предоставляет интерфейс для прохождения тестов участниками.

## Компоненты

- `ParticipantTestingPage` - Основная страница тестирования участника

## Возможности

- Отображение назначенных тестов
- Прохождение тестов участниками
- Отправка ответов на тесты
- Просмотр промежуточных результатов

## Зависимости

- Модуль решателя (`src/modules/solver`)
- Solver API (`src/shared/api/solver`)

## Использование

Модуль доступен через личный кабинет участника и предоставляет функциональность для прохождения назначенных тестов.
