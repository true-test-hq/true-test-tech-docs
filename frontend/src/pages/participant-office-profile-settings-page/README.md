# Модуль настроек профиля участника

Этот модуль предоставляет интерфейс для управления настройками профиля участника.

## Компоненты

- `ParticipantOfficeProfileSettingsPage` - Страница настроек профиля участника

## Возможности

- Просмотр и редактирование личной информации участника
- Изменение настроек учетной записи
- Управление предпочтениями интерфейса
- Настройка уведомлений

## Зависимости

- Модуль аккаунта (`src/modules/account`)
- Auth API (`src/shared/api/auth`)

## Использование

Модуль доступен через личный кабинет участника и предоставляет функциональность для управления персональными настройками.
