# Модуль настроек профиля администратора

Этот модуль предоставляет интерфейс для управления настройками профиля администратора.

## Компоненты

- `AdminOfficeProfileSettingsPage` - Страница настроек профиля администратора

## Возможности

- Просмотр и редактирование личной информации администратора
- Изменение настроек учетной записи
- Управление предпочтениями интерфейса
- Настройка уведомлений

## Зависимости

- Модуль аккаунта (`src/modules/account`)
- Auth API (`src/shared/api/auth`)

## Использование

Модуль доступен через административную панель и предоставляет функциональность для управления персональными настройками администратора.
