# Публичные модули

Этот модуль содержит публичные страницы и компоненты системы, доступные без аутентификации.

## Компоненты

- `DemoCustomerExam` - Демонстрационный экзамен для клиентов
- `PreparingForExam` - Страница подготовки к экзамену
- `CheckDevice` - Компонент проверки устройства
- `DemoTest` - Демонстрационный тест
- `ExamRules` - Правила экзамена

## Возможности

- Демонстрация возможностей системы потенциальным клиентам
- Подготовка пользователей к экзаменам
- Проверка совместимости устройств
- Ознакомление с правилами проведения экзаменов

## Зависимости

- Модуль решателя (`src/modules/solver`)
- Solver API (`src/shared/api/solver`)

## Использование

Публичные модули используются для привлечения новых клиентов и подготовки пользователей к работе с системой. Доступны без аутентификации.
