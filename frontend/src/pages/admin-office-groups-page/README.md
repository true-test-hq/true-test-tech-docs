# Модуль администрирования групп

Этот модуль предоставляет интерфейс для управления группами пользователей в системе администрирования.

## Компоненты

- `AdminOfficeGroupsPage` - Основная страница управления группами

## Возможности

- Просмотр списка групп
- Создание новых групп
- Редактирование существующих групп
- Удаление групп
- Управление участниками групп

## Зависимости

- Groups API (`src/shared/api/groups`)
- Модуль групп (`src/modules/groups`)

## Использование

Модуль доступен через панель администратора и предоставляет функциональность для управления группами пользователей в системе.
