# Модуль редакторов

Этот модуль предоставляет интерфейсы для создания и редактирования различных типов материалов.

## Компоненты

- `AdaptivePlaylistEditorPage` - Редактор адаптивных плейлистов
- `LinearPlaylistEditorPage` - Редактор линейных плейлистов
- `TaskEditorPage` - Редактор задач

## Возможности

- Создание и редактирование задач
- Создание и редактирование плейлистов (адаптивных и линейных)
- Управление структурой материалов
- Настройка параметров прохождения материалов

## Технические детали

### Архитектура модуля

Модуль редакторов следует модульной архитектуре с четким разделением ответственности:
- `src/modules/materials/` - Бизнес-логика и хуки для работы с материалами
- `src/pages/editors/` - Компоненты страниц редакторов
- `src/shared/api/materials/` - API клиент для взаимодействия с бэкендом

### Хуки данных

Модуль предоставляет следующие хуки для работы с материалами:
- `useMaterialAsync` - Получение данных о материале по ID
- `useMaterialsAsync` - Получение списка материалов
- `useFoldersAsync` - Получение списка папок
- `useCreateFolderAsync` - Создание новой папки
- `useRenameFolderAsync` - Переименование папки
- `useRemoveFolderAsync` - Удаление папки
- `useRemoveMaterialAsync` - Удаление материала
- `useDuplicateMaterialAsync` - Дублирование материала
- `useRenameMaterialAsync` - Переименование материала
- `useStartMaterialAsync` - Запуск материала
- `useUpsertTaskAsync` - Создание/обновление задачи
- `useUpsertPlaylistAsync` - Создание/обновление плейлиста

### API интеграция

Модуль использует `MaterialsApi` для взаимодействия с бэкендом:
- `getMaterials` - Получение списка материалов
- `getMaterialById` - Получение материала по ID
- `createTask` - Создание задачи
- `updateTaskById` - Обновление задачи
- `createPlaylistAdaptive` - Создание адаптивного плейлиста
- `createPlaylistLinear` - Создание линейного плейлиста
- `updatePlaylistLinear` - Обновление линейного плейлиста
- `updatePlaylistAdaptive` - Обновление адаптивного плейлиста
- `removeOwnMaterialById` - Удаление материала
- `duplicateMaterial` - Дублирование материала
- `renameMaterial` - Переименование материала
- `startMaterial` - Запуск материала
- `getFolders` - Получение списка папок
- `createFolder` - Создание папки
- `renameFolder` - Переименование папки
- `removeFolder` - Удаление папки

### Типы данных

Модуль использует строгую типизацию для всех материалов:
- `IMaterialListItem` - Тип для элемента списка материалов
- `ITaskMaterial` - Тип для задачи
- `IPlaylistMaterial` - Тип для плейлиста
- `IRemoteFolder` - Тип для папки

## Зависимости

- Materials API (`src/shared/api/materials`)
- Модуль материалов (`src/modules/materials`)
- Composite UI компоненты (`src/shared/composite-ui`)
- React Query для управления состоянием данных

## Использование

Модуль используется администраторами для создания и редактирования учебных материалов, задач и плейлистов. Предоставляет визуальные редакторы для различных типов контента.
