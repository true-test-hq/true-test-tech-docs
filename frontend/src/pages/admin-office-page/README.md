# Модуль административной панели

Этот модуль предоставляет основной интерфейс для административной панели системы.

## Компоненты

- `AdminOfficePage` - Основная страница административной панели

## Возможности

- Навигация по разделам администрирования
- Доступ к модулям управления пользователями, группами, материалами и событиями
- Просмотр общей статистики системы

## Зависимости

- Модули администрирования (`src/pages/admin-office-*`)
- Модуль обзора (`src/modules/overview`)

## Использование

Основная точка входа в административную панель системы. Предоставляет доступ ко всем функциям администрирования.
