# Модуль обзора системы

Этот модуль предоставляет общий обзор системы для администраторов.

## Компоненты

- `AdminOverviewPage` - Основная страница обзора системы

## Возможности

- Просмотр общей статистики системы
- Отображение ключевых метрик
- Доступ к основным разделам администрирования
- Быстрый доступ к часто используемым функциям

## Зависимости

- Модуль обзора (`src/modules/overview`)
- Overview API (`src/shared/api/overview`)

## Использование

Модуль служит стартовой страницей для администраторов и предоставляет общий обзор состояния системы.
