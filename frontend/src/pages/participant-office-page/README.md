# Модуль личного кабинета участника

Этот модуль предоставляет интерфейс личного кабинета для участников системы.

## Компоненты

- `ParticipantOfficePage` - Основная страница личного кабинета участника

## Возможности

- Просмотр личной информации участника
- Доступ к назначенным событиям и тестам
- Просмотр истории прохождения тестов
- Доступ к результатам и статистике

## Зависимости

- Модуль участника (`src/modules/participant`)
- Модуль решателя (`src/modules/solver`)

## Использование

Модуль служит основной точкой входа для участников системы и предоставляет доступ ко всем персональным данным и функциям.
