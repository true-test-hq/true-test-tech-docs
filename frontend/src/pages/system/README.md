# Модуль системы

Этот модуль предоставляет функциональность для мониторинга и управления системой.

## Компоненты

- `SystemHealthPage` - Страница состояния системы

## Возможности

- Мониторинг состояния системы
- Просмотр метрик производительности
- Проверка доступности сервисов
- Мониторинг нагрузки на систему

## Зависимости

- System API (`src/shared/api/system`)
- Модуль уведомлений (`src/modules/notifications`)

## Использование

Модуль используется администраторами для мониторинга состояния системы и диагностики проблем. Предоставляет информацию о здоровье системы и производительности.
