# Модуль управления участниками

Этот модуль предоставляет интерфейс для управления участниками системы в административной панели.

## Компоненты

- `AdminOfficeParticipantsPage` - Основная страница управления участниками

## Возможности

- Просмотр списка участников
- Поиск и фильтрация участников
- Редактирование профилей участников
- Блокировка/разблокировка участников
- Управление доступом участников к событиям

## Зависимости

- Модуль участников (`src/modules/participant`)
- Overview API (`src/shared/api/overview`)

## Использование

Модуль доступен через административную панель и предоставляет функциональность для управления всеми участниками системы.
