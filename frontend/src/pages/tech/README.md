# Технические модули

Этот модуль содержит технические страницы и компоненты системы.

## Компоненты

- `NotFoundPage` - Страница ошибки 404
- `MobileStubPage` - Заглушка для мобильных устройств
- `RootPage` - Корневая страница приложения
- `SocialAuthPage` - Страница социальной аутентификации

## Возможности

- Обработка ошибок 404
- Отображение заглушек для неподдерживаемых устройств
- Корневая маршрутизация приложения
- Обработка социальной аутентификации

## Зависимости

- Модуль аккаунта (`src/modules/account`)
- Auth API (`src/shared/api/auth`)

## Использование

Технические модули используются для обработки специфических случаев и обеспечения корректной работы приложения в различных условиях.
