# Модуль аналитики

Этот модуль предоставляет функциональность для анализа данных системы и результатов тестирования.

## Компоненты

- `AdminAnalyticsEventsAnswersDiversityPage` - Страница анализа разнообразия ответов на события

## Возможности

- Анализ разнообразия ответов участников
- Просмотр статистики проходимости тестов
- Анализ эффективности материалов
- Просмотр графиков и диаграмм
- Экспорт аналитических данных

## Технические детали

### Архитектура модуля

Модуль аналитики следует модульной архитектуре с четким разделением ответственности:
- `src/modules/analytics/model/` - Бизнес-логика и хуки для работы с данными
- `src/modules/analytics/ui/` - Компоненты пользовательского интерфейса
- `src/shared/api/analytics/` - API клиент для взаимодействия с бэкендом

### Хуки данных

Модуль предоставляет следующие хуки для работы с аналитическими данными:
- `useAnalyticsSolvabilityAsync` - Получение данных о решаемости задач
- `useAnalyticsEventAnswersDiversity` - Анализ разнообразия ответов на события
- `useApplyAdditionalAnswersEvent` - Применение дополнительных ответов к событию

### API интеграция

Модуль использует `AnalyticsApi` для взаимодействия с бэкендом:
- `getSolvability` - Получение данных о решаемости задач
- `getAnalyticsEventAnswersDiversity` - Получение данных о разнообразии ответов
- `applyAdditionalAnswersEvent` - Применение дополнительных ответов к событию

### Типы данных

Модуль использует строгую типизацию для всех аналитических данных:
- `IAnalyticsSolvability` - Тип для данных о решаемости
- `IAnalyticsEventAnswersDiversity` - Тип для данных о разнообразии ответов

## Зависимости

- Модуль аналитики (`src/modules/analytics`)
- Analytics API (`src/shared/api/analytics`)
- Модуль обзора (`src/modules/overview`)
- React Query для управления состоянием данных

## Использование

Модуль используется администраторами для анализа эффективности материалов и событий в системе. Предоставляет визуализацию данных и статистические отчеты.
