# Виджет предварительного просмотра материалов

Этот виджет предоставляет функциональность для предварительного просмотра материалов.

## Компоненты

- `MaterialPreviewWidget` - Основной компонент виджета

## Возможности

- Предварительный просмотр материалов
- Отображение метаинформации о материале
- Просмотр содержимого материалов

## Зависимости

- Materials API (`src/shared/api/materials`)
- Модуль материалов (`src/modules/materials`)

## Использование

Виджет используется в различных частях приложения для отображения предварительного просмотра материалов. Может быть интегрирован в списки материалов, редакторы и другие компоненты.
