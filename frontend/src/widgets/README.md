# Виджеты

Этот модуль содержит переиспользуемые компоненты (виджеты) для различных частей приложения.

## Виджеты

### MaterialPreviewWidget
Виджет предварительного просмотра материалов

### MaterialUpsertWidget
Виджет создания и редактирования материалов

### SolverSummaryWidget
Виджет сводки результатов решения тестов

### SolverWidget
Основной виджет для прохождения тестов

## Технические детали

### Архитектура виджетов

Виджеты следуют модульной архитектуре с четким разделением ответственности:
- `src/widgets/*/` - Каталог каждого виджета
- Каждый виджет содержит свои компоненты, хуки и стили
- Виджеты используют модули приложения для получения данных
- Виджеты используют shared компоненты для UI элементов

### Структура виджета

Каждый виджет следует стандартной структуре:
- `index.ts` - Экспорт компонентов виджета
- `WidgetName.tsx` - Основной компонент виджета
- `hooks/` - Хуки, специфичные для виджета
- `components/` - Компоненты, специфичные для виджета
- `*.module.scss` - Стили, специфичные для виджета

### Интеграция с модулями

Виджеты интегрируются с модулями приложения следующим образом:
- Используют хуки из `src/modules/*` для получения данных
- Используют типы из `src/modules/*/types` для типизации
- Не содержат бизнес-логики, только отображение данных

### Переиспользуемость

Виджеты спроектированы для максимальной переиспользуемости:
- Принимают данные через props
- Не зависят от конкретных страниц
- Могут быть встроены в любую часть приложения
- Следуют принципам компонентной архитектуры

## Зависимости

- React компоненты
- Модули приложения (`src/modules/*`)
- Shared компоненты (`src/shared/*`)
- React Query для управления состоянием данных

## Использование

Виджеты используются в различных частях приложения для обеспечения единообразного интерфейса и переиспользуемости кода. Каждый виджет инкапсулирует определенную функциональность и может быть легко интегрирован в любую часть приложения.
