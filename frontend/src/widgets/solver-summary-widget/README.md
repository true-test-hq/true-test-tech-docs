# Виджет сводки результатов решения тестов

Этот виджет предоставляет функциональность для отображения сводки результатов решения тестов.

## Компоненты

- `SolverSummaryWidget` - Основной компонент виджета сводки
- `EventSessionSummaryWidget` - Виджет сводки сессии события
- Компоненты визуализации результатов (статистика, графики, таблицы)

## Возможности

- Отображение сводки результатов тестирования
- Визуализация статистики прохождения тестов
- Отображение обратной связи от проверяющих
- Просмотр решений задач

## Зависимости

- Модуль решателя (`src/modules/solver`)
- Solver API (`src/shared/api/solver`)
- Модуль аналитики (`src/modules/analytics`)

## Использование

Виджет используется для отображения результатов тестирования участникам и администраторам. Предоставляет подробную информацию о прохождении тестов и результатах.
