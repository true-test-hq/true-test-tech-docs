# Виджет создания и редактирования материалов

Этот виджет предоставляет функциональность для создания и редактирования материалов.

## Компоненты

- `MaterialUpsertWidget` - Основной компонент виджета

## Возможности

- Создание новых материалов
- Редактирование существующих материалов
- Валидация данных материалов
- Сохранение материалов

## Зависимости

- Materials API (`src/shared/api/materials`)
- Модуль материалов (`src/modules/materials`)

## Использование

Виджет используется в редакторах материалов для предоставления унифицированного интерфейса создания и редактирования материалов различных типов.
