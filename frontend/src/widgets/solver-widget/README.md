# Виджет решения тестов

Этот виджет предоставляет основную функциональность для прохождения тестов.

## Компоненты

- `SolverWidget` - Основной компонент виджета решения тестов
- Компоненты управления процессом тестирования (таймер, навигация, кнопки управления)
- Компоненты отображения заданий
- Компоненты прокторинга

## Возможности

- Прохождение тестов различного типа
- Управление временем тестирования
- Навигация по заданиям
- Система прокторинга для контроля честности прохождения
- Сохранение промежуточных результатов

## Технические детали

### Архитектура виджета

Виджет решения тестов следует компонентной архитектуре:
- `src/widgets/solver-widget/` - Основной каталог виджета
- `components/` - Компоненты управления процессом тестирования
- `hooks/` - Хуки для работы с данными решателя
- Использует хуки из `src/modules/solver/` для получения данных

### Основные компоненты

- `SolverWidget` - Основной компонент, управляющий процессом тестирования
- `SolverStepper` - Компонент навигации по шагам теста
- `SolverTimer` - Компонент отображения времени тестирования
- `SolverSessionInfo` - Компонент отображения информации о сессии
- `ProctoringGuard` - Компонент системы прокторинга
- `SolverConfirmation` - Компонент подтверждения завершения теста

### Хуки данных

Виджет использует следующие хуки из модуля решателя:
- `useSolverConfigAsync` - Получение конфигурации сессии
- `useSolverTaskAsync` - Получение задачи для текущей сессии
- `useSubmitSolutionSolverAsync` - Отправка решения
- `useMoveSolverSession` - Перемещение по сессии
- `useFinishSolverSession` - Завершение сессии
- `useSolverAudioRetries` - Получение количества попыток воспроизведения аудио

### Интеграция с API

Виджет интегрируется с `SolverApi` через хуки модуля решателя:
- `getSessionConfig` - Получение конфигурации сессии
- `getSessionTask` - Получение задачи для текущей сессии
- `submitSolution` - Отправка решения
- `moveSolverSession` - Перемещение по сессии
- `finishSolverSession` - Завершение сессии

### Типы данных

Виджет использует строгую типизацию для всех данных:
- `ISolverSessionConfig` - Тип для конфигурации сессии
- `ISolverSessionState` - Тип для состояния сессии
- `ISolverTask` - Тип для задачи
- `ISolverSolution` - Тип для решения

## Зависимости

- Модуль решателя (`src/modules/solver`)
- Solver API (`src/shared/api/solver`)
- Модуль аккаунта (`src/modules/account`)
- React Query для управления состоянием данных

## Использование

Виджет используется в сессиях тестирования для предоставления интерфейса прохождения тестов участникам. Является основным компонентом в процессе тестирования.
